# Sequential  Access

連續的讀檔案。

需要： current position，read or write 的長度。

reset: 移動到別的位置

rewind: 移動到初始位置

ex: C 中的 `fwrite`, `fopen`

file 存在 disk ，而 disk 用 sequential access 比較快

# Direct (random) access

直接告訴檔案系統，要讀 file 的什麼位置。

要告訴他： 要讀哪個 position。 

## 缺點

假如 User 就是要連續讀整個 file， code 會變得比較麻煩

## 優點

比較彈性

# 變形： Index Access

例如:  Database 中，search 某一筆資料。
